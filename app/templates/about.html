About

WebDoI is a tool that wraps a frontend around
the HowDoI tool which allows you to find coding
examples easier, without the use of google.

Created by @maxbridgland as a small side project.
Hope you find it useful!
Uses the Equinox Bootstrap Theme

How To Find Your Results The Best

If you want to find a result easily you must
use a well formed query. A well formed
query uses descriptive keywords to find the
most relevant results. Here are a few examples:
